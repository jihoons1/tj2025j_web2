플로터
1. 설치
    1) https://flutter.dev/
    2) [Get started]
        Install and set up Flutter
        -> If you prefer to manually install Flutter, follow the instructions in Install Flutter manually.
        -> Install and set up Flutter 3.35.7 zip 다운로드
        다운로드 폴더 flutter > C://src 폴더 생성후 src 로 fluter 폴더 이동(넣기)
    2. 환경변수 ( 개발환경 없이도 윈도우에서 플러터 실행 )
        1) 윈도우 검색창에 '시스템 환경 변수 편집'
        2) 고급 -> 환경변수 -> 사용자 변수 ->  path 변수를 더블클릭
        3. 새로만들기 클릭후 플러터를 설치한 폴더내 bin 폴더의 경로 작성
            예] C:\src\flutter\bin
        확인] cmd -> flutter doctor 입력
            [v] Flutter 출력되면 성공
